Перед проверкой практической работы установите редактор изображение ImageMagick (инсталятор прилагается)
**
**
*****ПРАВИЛА УСТАНОВКИ*******
*
Во время установки, отметьте чекбоксы:
 - Add application directory to your system path
 - Install legacy utilites (e.g. convert)
**
**
*****ПРАВИЛА ИСПОЛЬЗОВАНИЯ*****
*
В файле index.php написан требующийся класс с описанием его методов.
Двое из них закомментированы. При  раскоменчивании метода комментируйте остальные,
иначе сработает только первый.
*
Если вы хотите проверить действие скрипта на своем изображении,
загрузите его в корень проекта под именем "example", удалив предыдущий файл.
Форматом загружаемоего изображения должен быть jpg.
