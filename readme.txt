Перед проверкой практической работы установите редактор изображение ImageMagick (инсталятор прилагается)
**
**
**
**
*****ПРАВИЛА УСТАНОВКИ*******
*
Во время установки, отметьте чекбоксы:
 - Add application directory to your system path
 - Install legacy utilites (e.g. convert)
*
**
**
**
*****ПРАВИЛА ИСПОЛЬЗОВАНИЯ*****
*
 1. В файле index.php написан требующийся класс с описанием его методов.
Двое из них закомментированы. При  раскоменчивании метода комментируйте остальные,
иначе сработает только первый.
*
 2. К практической проилагается набор картинок формтаов jpg, png и gif. 
Одна из них используются по умолчанию. Для изменения картинки в 
передаваемом конструктору параметре просто меняейте расширение изображения.
*
 3. Если вы хотите проверить действие скрипта на своем изображении,
загрузите его в корень проекта и в параметре конструктора напишите его название с указанием формата.
